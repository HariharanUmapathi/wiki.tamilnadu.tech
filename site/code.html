  • Tamil Lemmatizer

    Tamil Lemmatizer is a character-level sequence model that converts inflected Tamil words into their base (lemma) form. It is built using PyTorch and supports lemmatization of both verbs and nouns. The model generalizes well even for unseen words through character-level encoding.
